## Summary

Clone the source code locally

```bash
cd Desktop/ && git clone https://github.com/axjtan/https://github.com/axjtan/LoginNSignupExample.git
cd LoginNSignupExample/ && npm install
```

Run on Android:

```bash
react-native run-android
```
